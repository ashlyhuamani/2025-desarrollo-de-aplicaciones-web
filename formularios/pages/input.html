Elementos de inputs

inicio inpust buttons

los inputs son los elemntos que se crean un conetenido de un formulario, y nos permiten interactuar con el usuario, recolectar informacion de nuestra pagina

TAREA: averiguar los tipos de input

- Caja de texto para que el usuario introduzca texto. - Caja de texto para introducir contraseñas (el texto se muestra como asteriscos u otro carácter oculto). - Caja de texto para introducir números. - Caja de texto para introducir direcciones de correo electrónico. - Casilla de verificación que permite al usuario seleccionar una o varias opciones. - Botón de opción que permite al usuario seleccionar una sola opción de un grupo de opciones. - Caja de texto desplegable para seleccionar una fecha. - Botón para cargar archivos desde el dispositivo del usuario. - Botón para enviar el formulario. - Botón para restablecer el formulario a sus valores predeterminados.
